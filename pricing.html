Frequently Asked Questions

What's the difference between Free and Pro?

Free includes all deterministic security rules and basic scanning. Pro adds AI-powered analysis via Groq, unlimited scans, cloud history, and GitHub integration. The AI provides more nuanced analysis and better false-positive detection.

Do I need a Groq API key?

If you're on the Free plan, no — deterministic rules work without any API key. For Pro/Team, we handle the Groq integration for you. If you want to self-host with your own Groq key, that's also supported.

Is my data secure?

Yes. Scans are processed in memory and not permanently stored unless you're signed in and on a paid plan. We don't train on your data. For sensitive data, use redacted or staging samples.

Can I use this in CI/CD?

Yes! The GitHub Action is available for all plans. Free plan works great for public repos. Team plan includes private repo support and API access for custom integrations.

What happens if I hit the rate limit?

Free plan allows 60 scans per 15 minutes. If you need more, upgrade to Pro for unlimited scans. Rate limits help us prevent abuse and keep the service fast for everyone.
